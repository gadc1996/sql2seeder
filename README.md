# SQL 2 Laraval Seeder

## Description

Utility script to convert sql files to laravel seeder

# Setup

```
git clone https://github.com/gadc1996/sql2seeder.git
cd sql2seeder
```

Export the tables you want to convert and put them inside the sql directory

## Usage

```
python main.py
```
